# Лабораторна робота 1

## Тема
Основи C#, об’єктно-орієнтоване програмування, проєктування класів, робота з колекціями, багатошарова архітектура, консольний застосунок.

## Мета
Розробити базову об’єктну модель застосунку, реалізувати штучне сховище даних, сервіс взаємодії зі сховищем та консольний інтерфейс для перегляду сутностей.

## Реалізовано

- Дві сутності:
  - Сутність 1-го рівня — **Кафедра**
  - Сутність 2-го рівня — **Викладач**
- Enum для кожної сутності
- Окремі класи для зберігання даних
- Окремі класи для створення, відображення та редагування
- Штучне сховище з реалістичними початковими даними
- Сервіс роботи зі сховищем
- Дотримання принципу **Single Responsibility**
- Розподіл проєктів відповідно до багатошарової архітектури

## Структура рішення

- `University.Models` — моделі даних (сутності, enum)
- `University.Services` — штучне сховище та сервіс доступу до даних
- `University.Presentation` — логіка представлення
- `University.ConsoleApp` — консольний застосунок

## Запуск ЛР1

```bash
dotnet run --project University.ConsoleApp# Програмування на C# — Лабораторні роботи
```

# Лабораторна робота 2

## Тема
Розробка UI-застосунку, навігація між сторінками, Inversion of Control, Dependency Injection.

## Технологія
.NET MAUI (Android), реалізація через Code-Behind (без використання MVVM).

## Мета
Створити графічний застосунок на основі моделей та сервісів ЛР1 із реалізацією навігації між сторінками та використанням Dependency Injection.

## Реалізовано

-Один головний Window (відповідно до вимог)
- 3 сторінки:
  - Список кафедр
  - Деталі кафедри з переліком викладачів
  - Деталі викладача
- Можливість повернення назад
- Використання інтерфейсу `IUniversityStorageService`
- Реєстрація сервісів через DI-контейнер у `MauiProgram.cs`
- Повторне використання моделей та сервісів з ЛР1


## Структура рішення

- `University.UI` — MAUI UI-застосунок

## Запуск ЛР2
Запустити Android Emulator через Android Studio

У корені репозиторію виконати:

```bash
dotnet build University.UI -f net10.0-android
dotnet run --project University.UI -f net10.0-android
```
